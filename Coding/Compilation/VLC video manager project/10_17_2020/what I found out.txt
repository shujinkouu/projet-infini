one can control python using the keyboard usng the module keyboard

by using keyboard.read_hotkey(suppress=True) in python the user can find out what a key is called

keyboard.is_pressed('q')
this allow to know when q is pressed !

par exemple
while True:
    try:
        if keyboard.is_pressed('q'):
            print('q ete presse')
            break
    except:
        break
